# Гигиена артикуляций

## Назначение

Документ фиксирует **протокол операторной факторизации** всех «само-X» / «self-X» / «auto-X» конструкций в корпусе УГМ. Протокол — не стилистическое предпочтение, а **структурный инвариант**, закреплённый теоремой **NO-19 (Articulation Hygiene)** в Noesis ([diakrisis.gst.st/11-noesis/07-theorems](http://diakrisis.gst.st/11-noesis/07-theorems)). Протокол обязателен для любой артикуляции, проходящей core-pipeline Noesis (SMT → Axi → AFN-T → **Hygiene**).

## Принцип

Любая «само-X» конструкция в артикуляции — **недоопределена** без явной операторной факторизации. Не сказано:

- *какой* оператор $\Phi$ задаёт рекурсивное/рефлексивное действие,
- *какая* траектория $\Phi^\kappa$ (ординал $\kappa$) порождает конструкцию,
- *какой* терминальный/фиксированный объект $t$ замыкает её.

**Требование гигиены**: для каждой поверхностной «само-X» в артикуляции должна быть определена явная троица $(\Phi, \Phi^\kappa, t)$. Без неё артикуляция отвергается Hygiene-этапом pipeline.

## Философский контекст

Протокол формализует наблюдение Е. Чурилова (anticomplexity.org) в рамках метастемологической программы: конструкции с «само-» синтаксически подозрительны, поскольку смешивают описующую и описываемую позиции в одном знаке. Решение: явно развести эти позиции через оператор + траекторию + терминал.

В рамках [Diakrisis](http://diakrisis.gst.st) / [Актика](http://diakrisis.gst.st/actic) эта дисциплина уже встроена на уровне канонического примитива (эндофункторы $\mathsf{M}$ и $\mathsf{A}$ с accessibility, ординальная $\nu/\varepsilon$-стратификация, $\mathrm{Fix}$-множества). NO-19 поднимает её из имплицитного формального свойства в явный инвариант core-pipeline [Noesis](http://diakrisis.gst.st/noesis).

### Что **не** подлежит замене (стандартные технические термины)

- **self-adjoint** (эрмитов) — фиксированный математический термин, не рефлексивный оборот.
- **self-attention** (механизм Transformer-ов) — ML-термин, сохранять.
- **self-assembly** (в химии / нанотехнологиях) — физический термин, сохранять.
- **self-similarity** (фрактальная) — математический термин, сохранять.

Критерий: если «само-» выражает **свойство отображения/объекта**, определённое независимо от рефлексивной интерпретации, — термин сохраняется как есть.

## Применение в УГМ

### Уровень 1: концептуальные файлы (prose-rewrite)

Файлы, где «само-X» появляется в описательном смысле, **полностью** подлежат замене по таблице выше. Список высокоприоритетных:

- `/consciousness/foundations/self-observation.md` — канонический doc, рефакторится первым (см. §«Применённая факторизация» ниже).
- `/proofs/categorical/formalization-phi.md` — формализация $\varphi$, уже оператор-ориентирован, требует только сверки языка.
- `/core/foundations/consequences.md`, `axiom-septicity.md` — базовые следствия, поверхностный слой.
- `/consciousness/hierarchy/interiority-hierarchy.md`, `depth-tower.md` — иерархические структуры.
- `/proofs/consciousness/interiority-hierarchy.md`, `conscious-window.md` — доказательства.
- `/applied/coherence-cybernetics/theorems.md`, `definitions.md` — применения.

### Уровень 2: статус-регистр и сводные теоремы

`/reference/status-registry.md`, `/reference/specification.md` — описания теорем. «Само-» в заголовках теорем переписывается по протоколу; номера теорем сохраняются.

### Уровень 3: формальные proof-файлы

В доказательствах (`/proofs/*`) формальные определения **уже** операторные ($\varphi$, $\mathcal{L}_\Omega$, CPTP, T-96). Замена нужна только в промежуточных комментариях и motivation-параграфах.

## Применённая факторизация: канонический пример

Демонстрация протокола на ключевом концепте.

**До (риторика)**:
> УГМ устанавливает самомоделирование через оператор $\varphi$. Самонаблюдение — это способность системы описывать собственное состояние. Автопоэзис — самовоспроизводство системы через самомоделирование.

**После (операторно)**:
> УГМ устанавливает **категорное отображение** $\varphi: \mathrm{D}(\mathbb{C}^7) \to \mathrm{D}(\mathbb{C}^7)$, имеющее неподвижную точку $\rho^* = \varphi(\Gamma)$ (T-96). **Рефлексия** системы относительно собственного состояния формализуется как **terminal coalgebra**-структура на $\Gamma$-динамике с характеризующей мерой $R = 1/(7P)$ (T-126). **Замкнутость под $\varphi$-траекторией** (на дуальной AC-стороне — $\mathsf{A}$-фиксточка уровня $\omega^2$ по 113.T Diakrisis) — воспроизводство траектории без внешнего поддержания.

Операторное содержание — **идентично** риторическому. Но:
1. Каждый элемент явно типизирован ($\varphi$ — оператор, $\rho^*$ — состояние, $R$ — мера, итерация — траектория).
2. Описующая и описываемая позиции разведены ($\Gamma$ и $\varphi(\Gamma)$ — разные точки в $\mathrm{D}(\mathbb{C}^7)$, хотя и связаны уравнением неподвижной точки).
3. Протокол совместим с hygiene-проверкой Noesis (NO-19): факторизация явна.

## Примечание

Протокол не отменяет существующие теоремы УГМ — все T-1..T-223+ остаются формально корректными. Рефакторинг — **интерфейсный**, затрагивает только prose-слой. Формальное ядро ($\varphi$, $\mathcal{L}_\Omega$, T-96, T-190, T-223) **уже** оперирует в операторном языке; гигиена синхронизирует поверхность с ядром.
