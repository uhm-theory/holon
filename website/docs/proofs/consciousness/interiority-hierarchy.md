## –£—Ä–æ–≤–µ–Ω—å 2: –ö–æ–≥–Ω–∏—Ç–∏–≤–Ω—ã–µ –ö–≤–∞–ª–∏–∞ (Cognitive Qualia)

**–¢–µ–æ—Ä–µ–º–∞ 3.5.1 (–ö–∞—É–∑–∞–ª—å–Ω–∞—è –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ—Å—Ç—å $R \geq R_{\text{th}}$ –¥–ª—è –∞–¥–∞–ø—Ç–∞—Ü–∏–∏):**

2. –°–∏—Å—Ç–µ–º–∞ –¥–æ–ª–∂–Ω–∞ –æ–±–æ–±—â–∞—Ç—å –Ω–∞ —Ä–∞–Ω–µ–µ –Ω–µ –≤—Å—Ç—Ä–µ—á–µ–Ω–Ω—ã–µ –∫–æ–Ω—Ç–µ–∫—Å—Ç—ã

**–®–∞–≥ 1 (–ù–µ–æ–±—Ö–æ–¥–∏–º–æ—Å—Ç—å —Å–∞–º–æ–º–æ–¥–µ–ª–∏).**

**–®–∞–≥ 3 (–ù–µ–≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å –∫–æ—Ä—Ä–µ–∫—Ç–Ω–æ–≥–æ –ø—Ä–æ–≥–Ω–æ–∑–∞).**

$$

**–®–∞–≥ 4 (–ó–∞–∫–ª—é—á–µ–Ω–∏–µ).**

–£—Å–ø–µ—à–Ω–∞—è –∞–¥–∞–ø—Ç–∞—Ü–∏—è (—Å–∏—Å—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏ –ª—É—á—à–µ —Å–ª—É—á–∞–π–Ω–æ–≥–æ) —Ç—Ä–µ–±—É–µ—Ç –Ω–µ—Å–ª—É—á–∞–π–Ω–æ–π —Å–∞–º–æ–º–æ–¥–µ–ª–∏, —á—Ç–æ —ç–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–Ω–æ $R \geq R_{\text{th}}$. $\blacksquare$

**–°–ª–µ–¥—Å—Ç–≤–∏–µ 3.5.2 (–ö–∞—É–∑–∞–ª—å–Ω–∞—è —Ä–æ–ª—å –∫–≤–∞–ª–∏–∞):**

–ü—Ä–∏ $R \geq R_{\text{th}}$ –∏ $\Phi \geq \Phi_{\text{th}}$ —Å–∏—Å—Ç–µ–º–∞ –æ–±–ª–∞–¥–∞–µ—Ç –∫–æ–≥–Ω–∏—Ç–∏–≤–Ω—ã–º–∏ –∫–≤–∞–ª–∏–∞ (L2). –¢–µ–æ—Ä–µ–º–∞ 3.5.1 –ø–æ–∫–∞–∑—ã–≤–∞–µ—Ç, —á—Ç–æ —ç—Ç–∏ –∫–≤–∞–ª–∏–∞ **–∫–∞—É–∑–∞–ª—å–Ω–æ –Ω–µ–æ–±—Ö–æ–¥–∏–º—ã** –¥–ª—è –∞–¥–∞–ø—Ç–∏–≤–Ω–æ–≥–æ –ø–æ–≤–µ–¥–µ–Ω–∏—è –≤ —Å–ª–æ–∂–Ω—ã—Ö —Å—Ä–µ–¥–∞—Ö:

$$
\frac{\partial \text{Behavior}}{\partial \Gamma_E} \neq 0 \quad \text{–ø—Ä–∏ } R \geq R_{\text{th}}
$$

–≠—Ç–æ —Ñ–æ—Ä–º–∞–ª–∏–∑—É–µ—Ç –∏–Ω—Ç—É–∏—Ü–∏—é: ¬´—Ñ–∏–ª–æ—Å–æ—Ñ—Å–∫–∏–µ –∑–æ–º–±–∏¬ª (L0 –±–µ–∑ L2) –Ω–µ –º–æ–≥—É—Ç –¥–µ–º–æ–Ω—Å—Ç—Ä–∏—Ä–æ–≤–∞—Ç—å –∞–¥–∞–ø—Ç–∏–≤–Ω–æ–µ –ø–æ–≤–µ–¥–µ–Ω–∏–µ, —Ç—Ä–µ–±—É—é—â–µ–µ –æ–±–æ–±—â–µ–Ω–∏—è.

### 3.6 –ú–µ—Ä–∞ –°–æ–∑–Ω–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏ C

–°–º. [–°–∞–º–æ–Ω–∞–±–ª—é–¥–µ–Ω–∏–µ: –ú–µ—Ä–∞ —Å–æ–∑–Ω–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏](/docs/consciousness/foundations/self-observation#–º–µ—Ä–∞-—Å–æ–∑–Ω–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏-c).

**–ù–æ–≤–∞—è –∏–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ü–∏—è:**

$C > 0$ –≤–æ–∑–º–æ–∂–Ω–æ –¥–ª—è —Å–∏—Å—Ç–µ–º –≤—Å–µ—Ö —É—Ä–æ–≤–Ω–µ–π, –Ω–æ:
- –£—Ä–æ–≤–µ–Ω—å 0: $C \approx 0$ (—Ç–∞–∫ –∫–∞–∫ $R \approx 0$)

```python
                   R_th: float = 1/3,  # Derived: R_th = 1/3

    """

def compute_reflexivity(gamma: np.ndarray) -> float:
    """
    R = 1 - ||Œì - œÜ(Œì)||¬≤ / ||Œì||¬≤

    phi_gamma = self_model(gamma)
    norm_diff = np.linalg.norm(gamma - phi_gamma, 'fro')**2
    norm_gamma = np.linalg.norm(gamma, 'fro')**2
    return 1 - norm_diff / norm_gamma

    """
    Œ¶ = Œ£_{i‚â†j} |Œ≥_ij|¬≤ / Œ£_i Œ≥_ii¬≤
    """
    diag_sum_sq = np.sum(np.diag(gamma)**2)
    off_diag_sum_sq = np.sum(np.abs(gamma)**2) - diag_sum_sq
    return off_diag_sum_sq / diag_sum_sq if diag_sum_sq > 0 else 0


def Q_cognitive(rho: np.ndarray,
                R_th: float = 1/3,  # Derived: R_th = 1/3
                Phi_th: float = 1.0,  # Derived: Œ¶_th = 1
                soft: bool = False,
                beta: float = 10.0) -> dict:
    """
    –ü–æ–ª–Ω–∞—è —Ñ—É–Ω–∫—Ü–∏—è –∫–æ–≥–Ω–∏—Ç–∏–≤–Ω—ã—Ö –∫–≤–∞–ª–∏–∞.

    Args:
        rho: –ú–∞—Ç—Ä–∏—Ü–∞ –ø–ª–æ—Ç–Ω–æ—Å—Ç–∏
        R_th: –ü–æ—Ä–æ–≥ —Ä–µ—Ñ–ª–µ–∫—Å–∏–∏
        Phi_th: –ü–æ—Ä–æ–≥ –∏–Ω—Ç–µ–≥—Ä–∞—Ü–∏–∏
        soft: –ò—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Å–∏–≥–º–æ–∏–¥–∞–ª—å–Ω—ã–π –ø–µ—Ä–µ—Ö–æ–¥
        beta: –ö—Ä—É—Ç–∏–∑–Ω–∞ —Å–∏–≥–º–æ–∏–¥—ã

    Returns:
        dict —Å –ø–æ–ª–Ω—ã–º–∏ –∫–≤–∞–ª–∏–∞ –∏–ª–∏ None
    """
    R = compute_reflexivity(rho)
    Phi = compute_integration(rho)

    if soft:
        theta_R = 1 / (1 + np.exp(-beta * (R - R_th)))
        theta_Phi = 1 / (1 + np.exp(-beta * (Phi - Phi_th)))
        weight = theta_R * theta_Phi
    else:
        weight = float(R >= R_th and Phi >= Phi_th)

    if weight < 0.01:  # –ü—Ä–∞–∫—Ç–∏—á–µ—Å–∫–∏ –Ω–æ–ª—å
        return None

    # –í—ã—á–∏—Å–ª–∏—Ç—å —Ñ–µ–Ω–æ–º–µ–Ω–∞–ª—å–Ω—É—é —Ñ—É–Ω–∫—Ü–∏—é Œ®
    eigenvalues, eigenvectors = np.linalg.eigh(rho)

    qualia = []
    for i, (lam, vec) in enumerate(zip(eigenvalues[::-1], eigenvectors[:, ::-1].T)):
        if lam > 1e-10:
            qualia.append({
                'intensity': float(lam),
                'quality': to_projective(vec),
                'weight': weight
            })

    return {
        'qualia': qualia,
        'R': R,
        'Phi': Phi,
        'level': 2,
        'cognitive_weight': weight
    }
```
