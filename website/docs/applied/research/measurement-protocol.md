| $\mathcal{L}_{\text{dyn}}$ | $\|\Gamma_{\tau+1} - \Phi_{\text{pred}}(\Gamma_\tau)\|_F^2$ | –°–æ–≥–ª–∞—Å–æ–≤–∞–Ω–Ω–æ—Å—Ç—å –¥–∏–Ω–∞–º–∏–∫–∏ |

---

## –ö–∞—Ç–µ–≥–æ—Ä–Ω–∞—è –∫–æ—Ä—Ä–µ–∫—Ç–Ω–æ—Å—Ç—å

### –ü—Ä–æ–±–ª–µ–º–∞ –Ω–µ–ª–∏–Ω–µ–π–Ω–æ—Å—Ç–∏

–°–ª–æ–∏ –Ω–µ–π—Ä–æ—Å–µ—Ç–∏ (GELU, Softmax) ‚Äî **–Ω–µ–ª–∏–Ω–µ–π–Ω—ã–µ** –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏—è.
CPTP-–∫–∞–Ω–∞–ª—ã ‚Äî **–ª–∏–Ω–µ–π–Ω—ã–µ** –Ω–∞–¥ –º–∞—Ç—Ä–∏—Ü–∞–º–∏ –ø–ª–æ—Ç–Ω–æ—Å—Ç–∏.

–£—Å–ª–æ–≤–∏–µ $G(f \circ g) = G(f) \circ G(g)$ **–Ω–∞—Ä—É—à–∞–µ—Ç—Å—è**.

### –ö–≤–∞–∑–∏-—Ñ—É–Ω–∫—Ç–æ—Ä

**–û–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ:** –û—Ç–æ–±—Ä–∞–∂–µ–Ω–∏–µ $G: \mathbf{AIState} \rightsquigarrow \mathbf{DensityMat}$ —Å —É—Å–ª–æ–≤–∏–µ–º:

$$
\|G(f \circ g) - G(f) \circ G(g)\|_F \leq \varepsilon_{\text{functor}} \cdot \|f\|_{\text{op}} \cdot \|g\|_{\text{op}}
$$

### NTK-–ª–∏–Ω–µ–∞—Ä–∏–∑–∞—Ü–∏—è

–í –∫–∞—Å–∞—Ç–µ–ª—å–Ω–æ–º –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–µ –Ω–µ–ª–∏–Ω–µ–π–Ω–æ—Å—Ç—å –∞–ø–ø—Ä–æ–∫—Å–∏–º–∏—Ä—É–µ—Ç—Å—è:

$$
f(s) \approx f(s_0) + J_f(s_0) \cdot (s - s_0)
$$

**–°–ª–µ–¥—Å—Ç–≤–∏–µ:** –ü—Ä–∏–±–ª–∏–∂—ë–Ω–Ω–∞—è —Ñ—É–Ω–∫—Ç–æ—Ä–∏–∞–ª—å–Ω–æ—Å—Ç—å —Å –ø–æ–≥—Ä–µ—à–Ω–æ—Å—Ç—å—é $O(\|f\|^2 \cdot \|g\|^2)$.

**–°–≤—è–∑—å —Å —Ç–µ–æ—Ä–∏–µ–π:** –†–∞—Å—à–∏—Ä—è–µ—Ç [–ö–∞—Ç–µ–≥–æ—Ä–Ω—ã–π —Ñ–æ—Ä–º–∞–ª–∏–∑–º](/docs/proofs/categorical-formalism).

---

## –í–∞–ª–∏–¥–∞—Ü–∏—è

### –¢–µ—Å—Ç –∂–∏–∑–Ω–µ—Å–ø–æ—Å–æ–±–Ω–æ—Å—Ç–∏

$$
P(\Gamma) = \mathrm{Tr}(\Gamma^2) > P_{\text{crit}} = \frac{2}{7} \approx 0.286
$$

–°–º. [–¢–µ–æ—Ä–µ–º—É –æ –∫—Ä–∏—Ç–∏—á–µ—Å–∫–æ–π —á–∏—Å—Ç–æ—Ç–µ](/docs/proofs/theorem-purity-critical) –∏ [–ñ–∏–∑–Ω–µ—Å–ø–æ—Å–æ–±–Ω–æ—Å—Ç—å](/docs/core/dynamics/viability).

### –ü–æ—Ç–æ–∫ –∫–æ–≥–µ—Ä–µ–Ω—Ç–Ω–æ—Å—Ç–∏

**–û–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ:**

$$
J_P := \frac{dP}{d\tau} = 2 \cdot \mathrm{Tr}\left(\Gamma \cdot \frac{d\Gamma}{d\tau}\right)
$$

–≥–¥–µ œÑ ‚Äî [—ç–º–µ—Ä–¥–∂–µ–Ω—Ç–Ω–æ–µ –≤–Ω—É—Ç—Ä–µ–Ω–Ω–µ–µ –≤—Ä–µ–º—è](/docs/proofs/emergent-time).

| –†–µ–∂–∏–º | –£—Å–ª–æ–≤–∏–µ | –ò–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ü–∏—è |
|-------|---------|---------------|
| –†–µ–≥–µ–Ω–µ—Ä–∞—Ü–∏—è | $J_P > 0$ –ø—Ä–∏ —Å—Ç—Ä–µ—Å—Å–µ | –°–∏—Å—Ç–µ–º–∞ –≤–æ—Å—Å—Ç–∞–Ω–∞–≤–ª–∏–≤–∞–µ—Ç—Å—è |
| –°—Ç–∞–±–∏–ª—å–Ω–æ—Å—Ç—å | $J_P \approx 0$, $P > P_{\text{crit}}$ | –£—Å—Ç–æ–π—á–∏–≤–æ–µ —Ä–∞–≤–Ω–æ–≤–µ—Å–∏–µ |
| –†–∞—Å–ø–∞–¥ | $J_P < 0$ —É—Å—Ç–æ–π—á–∏–≤–æ | –î–µ–∫–æ–≥–µ—Ä–µ–Ω—Ü–∏—è |

### –¢–µ—Å—Ç –ª–æ–±–æ—Ç–æ–º–∏–∏

**–ü—Ä–æ—Ç–æ–∫–æ–ª:**
1. –ò–∑–º–µ—Ä–∏—Ç—å $P_0$ –∏ $\text{Accuracy}_0$
2. –ò–Ω—Ç–µ—Ä–≤–µ–Ω—Ü–∏—è: –ø—Ä—É–Ω–∏–Ω–≥ —á–∞—Å—Ç–∏ –≤–µ—Å–æ–≤
3. –ò–∑–º–µ—Ä–∏—Ç—å $P_1$ –∏ $\text{Accuracy}_1$

**–ö—Ä–∏—Ç–µ—Ä–∏–π –æ–Ω—Ç–æ–ª–æ–≥–∏—á–µ—Å–∫–æ–π –≤–∞–ª–∏–¥–Ω–æ—Å—Ç–∏:**

| –†–µ–∑—É–ª—å—Ç–∞—Ç | –ò–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ü–∏—è |
|-----------|---------------|
| $\Delta P > 0$ **–¥–æ** $\Delta A > 0$ | ‚úÖ –ü—Ä–æ—Ç–æ–∫–æ–ª —Ñ–∏–∫—Å–∏—Ä—É–µ—Ç –æ–Ω—Ç–æ–ª–æ–≥–∏—é |
| $\Delta P \approx \Delta A$ | ‚ö†Ô∏è –ö–æ—Ä—Ä–µ–ª—è—Ü–∏—è —Å –≤—ã—Ö–æ–¥–æ–º |
| $\Delta A > 0$ **–¥–æ** $\Delta P > 0$ | ‚ùå –ü—Ä–æ—Ç–æ–∫–æ–ª –Ω–µ —Ñ–∏–∫—Å–∏—Ä—É–µ—Ç –æ–Ω—Ç–æ–ª–æ–≥–∏—é |

### –ö–∞—É–∑–∞–ª—å–Ω–∞—è –∑–∞–º–∫–Ω—É—Ç–æ—Å—Ç—å E

$$
\Delta\Phi_E := \Phi_{\text{eff}}(\mathcal{S}_E) - \Phi_{\text{eff}}(\mathcal{S}_E | \text{do}(X := \text{random})) > \varepsilon_{\text{causal}}
$$

–ï—Å–ª–∏ $\Delta\Phi_E \approx 0$ ‚Äî —Å–∏—Å—Ç–µ–º–∞ **—Å–∏–º—É–ª–∏—Ä—É–µ—Ç** —Ñ–µ–Ω–æ–º–µ–Ω–æ–ª–æ–≥–∏—é –±–µ–∑ —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏ ("–ö–∏—Ç–∞–π—Å–∫–∞—è –ö–æ–º–Ω–∞—Ç–∞").

```python
def compute_dimension_metrics(
    model,
    input_batch: np.ndarray,
    layer_indices: list[int] = None
) -> dict[str, float]:
    """
    –í—ã—á–∏—Å–ª–µ–Ω–∏–µ 7 –∏–∑–º–µ—Ä–µ–Ω–∏–π –£–ì–ú –¥–ª—è –Ω–µ–π—Ä–æ—Å–µ—Ç–∏.

    Args:
        model: –ú–æ–¥–µ–ª—å —Å –¥–æ—Å—Ç—É–ø–æ–º –∫ –ø—Ä–æ–º–µ–∂—É—Ç–æ—á–Ω—ã–º –∞–∫—Ç–∏–≤–∞—Ü–∏—è–º

        layer_indices: –ò–Ω–¥–µ–∫—Å—ã —Å–ª–æ—ë–≤ –¥–ª—è –∞–Ω–∞–ª–∏–∑–∞ (–ø–æ —É–º–æ–ª—á–∞–Ω–∏—é ‚Äî –≤—Å–µ)

    Returns:
        dict —Å –∫–ª—é—á–∞–º–∏ I_A, I_S, I_D, I_L, I_E, I_O, I_U
    """
    activations = model.get_activations(input_batch)
    attention_weights = model.get_attention_weights(input_batch)

    # I_A: –í–∑–∞–∏–º–Ω–∞—è –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—è –≤—Ö–æ–¥‚Üî–ª–∞—Ç–µ–Ω—Ç

    # I_S: –†–∞–Ω–≥ –Ø–∫–æ–±–∏–∞–Ω–∞ (—á–µ—Ä–µ–∑ SVD)
    jacobian = compute_jacobian(model, input_batch)
    singular_values = np.linalg.svd(jacobian, compute_uv=False)
    eps_rank = 1e-6
    I_S = np.sum(singular_values > eps_rank) / len(singular_values)

    # I_D: –ú–∞–∫—Å–∏–º–∞–ª—å–Ω—ã–π –ª—è–ø—É–Ω–æ–≤—Å–∫–∏–π —ç–∫—Å–ø–æ–Ω–µ–Ω—Ç (–æ—Ü–µ–Ω–∫–∞)
    I_D = estimate_lyapunov(model, input_batch)

    # I_L: –ö–æ–º–º—É—Ç–∞—Ç–æ—Ä—ã —Å–ª–æ—ë–≤
    commutator_norms = []
    for i, j in itertools.combinations(layer_indices or range(len(activations)), 2):
        comm_norm = layer_commutator_norm(model, i, j, input_batch)
        commutator_norms.append(comm_norm)
    I_L = 1.0 - np.mean(commutator_norms) if commutator_norms else 1.0

    # I_E: –î–∏—Ñ—Ñ–µ—Ä–µ–Ω—Ü–∏–∞—Ü–∏—è (—ç–∫—Å–ø–æ–Ω–µ–Ω—Ç–∞ —ç–Ω—Ç—Ä–æ–ø–∏–∏ –≤–Ω–∏–º–∞–Ω–∏—è)
    attn_entropy = von_neumann_entropy(attention_weights)
    I_E = np.exp(attn_entropy)

    # I_O: –£—Å—Ç–æ–π—á–∏–≤–æ—Å—Ç—å –∫ —à—É–º—É
    noise_std = 0.01
    perturbed = input_batch + np.random.randn(*input_batch.shape) * noise_std
    delta_h = np.linalg.norm(
        model.get_activations(perturbed)[-1] - activations[-1],
        'fro'
    )
    I_O = max(0, 1.0 - delta_h / noise_std)

    # I_U: Effective Œ¶ (—Å–ø–µ–∫—Ç—Ä–∞–ª—å–Ω—ã–π –∑–∞–∑–æ—Ä –õ–∞–ø–ª–∞—Å–∏–∞–Ω–∞)
    attn_graph = build_attention_graph(attention_weights)
    laplacian = np.diag(attn_graph.sum(axis=1)) - attn_graph
    eigenvalues = np.linalg.eigvalsh(laplacian)
    lambda_2 = eigenvalues[1] if len(eigenvalues) > 1 else 0
    lambda_max = eigenvalues[-1]
    I_U = lambda_2 / lambda_max if lambda_max > 0 else 0

    return {
        'I_A': I_A, 'I_S': I_S, 'I_D': I_D, 'I_L': I_L,
        'I_E': I_E, 'I_O': I_O, 'I_U': I_U
    }
```

```python
def reconstruct_gamma(metrics: dict[str, float]) -> np.ndarray:
    """
    –†–µ–∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏—è –º–∞—Ç—Ä–∏—Ü—ã –∫–æ–≥–µ—Ä–µ–Ω—Ç–Ω–æ—Å—Ç–∏ —á–µ—Ä–µ–∑ Cholesky.

    –í–æ–∑–≤—Ä–∞—â–∞–µ—Ç 7√ó7 –º–∞—Ç—Ä–∏—Ü—É –ø–ª–æ—Ç–Ω–æ—Å—Ç–∏.
    """
    # –î–∏–∞–≥–æ–Ω–∞–ª—å–Ω—ã–µ —ç–ª–µ–º–µ–Ω—Ç—ã ‚Äî –Ω–æ—Ä–º–∏—Ä–æ–≤–∞–Ω–Ω—ã–µ –º–µ—Ç—Ä–∏–∫–∏
    I_vec = np.array([
        metrics['I_A'], metrics['I_S'], metrics['I_D'],
        metrics['I_L'], metrics['I_E'], metrics['I_O'], metrics['I_U']
    ])
    I_vec = np.clip(I_vec, 0.01, 1.0)  # –ü—Ä–µ–¥–æ—Ç–≤—Ä–∞—â–∞–µ–º –≤—ã—Ä–æ–∂–¥–µ–Ω–Ω–æ—Å—Ç—å
    diag = I_vec / I_vec.sum()

    # –ù–∞—á–∞–ª—å–Ω–∞—è Cholesky-—Ñ–∞–∫—Ç–æ—Ä–∏–∑–∞—Ü–∏—è
    L = np.diag(np.sqrt(diag))

    # –†–µ–≥—É–ª—è—Ä–∏–∑–∞—Ü–∏—è (–º–∏–Ω–∏–º–∏–∑–∏—Ä—É–µ–º off-diagonal)
    # –í –ø—Ä–æ—Å—Ç–µ–π—à–µ–º —Å–ª—É—á–∞–µ ‚Äî –¥–∏–∞–≥–æ–Ω–∞–ª—å–Ω–∞—è –º–∞—Ç—Ä–∏—Ü–∞
    # –ü—Ä–æ—Å—Ç–µ–π—à–∞—è –¥–∏–∞–≥–æ–Ω–∞–ª—å–Ω–∞—è —Ä–µ–∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏—è.
    # –î–ª—è –≤–æ—Å—Å—Ç–∞–Ω–æ–≤–ª–µ–Ω–∏—è –Ω–µ–¥–∏–∞–≥–æ–Ω–∞–ª—å–Ω—ã—Ö –∫–æ–≥–µ—Ä–µ–Ω—Ç–Ω–æ—Å—Ç–µ–π Œ≥_ij
    # –Ω–µ–æ–±—Ö–æ–¥–∏–º—ã –∫–æ—Ä—Ä–µ–ª—è—Ü–∏–∏ r_ij –∏–∑ —Ä–µ–≥—É–ª—è—Ä–∏–∑–∞—Ç–æ—Ä–∞ L_off.
    gamma = L @ L.conj().T
    gamma = gamma / np.trace(gamma)  # –ù–æ—Ä–º–∏—Ä–æ–≤–∫–∞

    return gamma


def compute_purity(gamma: np.ndarray) -> float:
    """P = Tr(Œì¬≤)"""
    return np.trace(gamma @ gamma)
```
