- **42D** ($\mathcal{H}_O \otimes \mathcal{H}_{6D} \cong \mathbb{C}^{42}$): –º–µ—Ö–∞–Ω–∏–∑–º Page-Wootters (—ç–º–µ—Ä–¥–∂–µ–Ω—Ç–Ω–æ–µ –≤—Ä–µ–º—è), –∫–∞–ª–∏–±—Ä–æ–≤–æ—á–Ω—ã–µ —Å–∏–º–º–µ—Ç—Ä–∏–∏ —ç–ª–µ–∫—Ç—Ä–æ—Å–ª–∞–±–æ–≥–æ —Å–µ–∫—Ç–æ—Ä–∞, —Ç–µ–Ω–∑–æ—Ä–Ω–∞—è –∑–∞–ø—É—Ç–∞–Ω–Ω–æ—Å—Ç—å.

**–†–∞–∑—Ä–µ—à—ë–Ω–Ω–∞—è —á–∞—Å—Ç—å [–¢]:** –¢–µ–Ω–∑–æ—Ä–Ω—ã–π —Ä–∞–∑—Ä—ã–≤ –¥–ª—è $\mathrm{Coh}_E$ –ø–æ–ª–Ω–æ—Å—Ç—å—é —Ä–∞–∑—Ä–µ—à—ë–Ω ‚Äî [C*-–∞–ª–≥–µ–±—Ä–∞–∏—á–µ—Å–∫–∞—è –ø—Ä–æ–µ–∫—Ü–∏—è –ì–∏–ª—å–±–µ—Ä—Ç–∞‚Äì–®–º–∏–¥—Ç–∞](/docs/core/foundations/axiom-septicity#hs-projection) –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç $\mathrm{Coh}_E$ –≤ 7D **—Ç–æ—á–Ω–æ**, –Ω–µ –ø—Ä–∏–±–µ–≥–∞—è –∫ —á–∞—Å—Ç–∏—á–Ω–æ–º—É —Å–ª–µ–¥—É. –û–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ –ø–æ–¥—Å–∏—Å—Ç–µ–º—ã —Ä–µ–∞–ª–∏–∑—É–µ—Ç—Å—è —á–µ—Ä–µ–∑ –≤–ª–æ–∂–µ–Ω–∏–µ C*-–ø–æ–¥–∞–ª–≥–µ–±—Ä—ã –∏ [—É—Å–ª–æ–≤–Ω–æ–µ –æ–∂–∏–¥–∞–Ω–∏–µ –£–º–µ–≥–∞–∫–∏](/docs/core/foundations/axiom-septicity#—Ç–µ–æ—Ä–µ–º–∞-—É—Å–ª–æ–≤–Ω–æ–µ-–æ–∂–∏–¥–∞–Ω–∏–µ). –≠—Ç–æ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã–π –∞–ø–ø–∞—Ä–∞—Ç –∞–ª–≥–µ–±—Ä–∞–∏—á–µ—Å–∫–æ–π –∫–≤–∞–Ω—Ç–æ–≤–æ–π —Ç–µ–æ—Ä–∏–∏ (Haag, 1996; Bratteli‚ÄìRobinson, 1987).

**–û—Ç–∫—Ä—ã—Ç–∞—è —á–∞—Å—Ç—å [–°]:** –ü–æ–ª–Ω–∞—è —Ä–µ–¥—É—Ü–∏—Ä–æ–≤–∞–Ω–Ω–∞—è –º–∞—Ç—Ä–∏—Ü–∞ $\rho_E = \mathrm{Tr}_{-E}(\Gamma)$ –∏ –º–µ—Ä–∞ –¥–∏—Ñ—Ñ–µ—Ä–µ–Ω—Ü–∏–∞—Ü–∏–∏ $D_{\text{diff}} = \exp(S_{vN}(\rho_E))$ –ø–æ-–ø—Ä–µ–∂–Ω–µ–º—É —Ç—Ä–µ–±—É—é—Ç —Ç–µ–Ω–∑–æ—Ä–Ω–æ–π —Ñ–∞–∫—Ç–æ—Ä–∏–∑–∞—Ü–∏–∏ (42D-—Ñ–æ—Ä–º–∞–ª–∏–∑–º), –ø–æ—Å–∫–æ–ª—å–∫—É —á–∞—Å—Ç–∏—á–Ω—ã–π —Å–ª–µ–¥ $\mathrm{Tr}_{\bar{E}}$ –Ω–µ –æ–ø—Ä–µ–¥–µ–ª—ë–Ω –≤ $\mathbb{C}^7$ (7 ‚Äî –ø—Ä–æ—Å—Ç–æ–µ —á–∏—Å–ª–æ). –£—Ç–≤–µ—Ä–∂–¥–µ–Ω–∏—è, –∏—Å–ø–æ–ª—å–∑—É—é—â–∏–µ $D_{\text{diff}}$, –∏–º–µ—é—Ç —Å—Ç–∞—Ç—É—Å **[–°]** ‚Äî —É—Å–ª–æ–≤–Ω—ã–µ –Ω–∞ 42D-—Ä–∞—Å—à–∏—Ä–µ–Ω–∏–µ.
:::
